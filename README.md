# Cybersecurity-Projects
Python-based cybersecurity toolkit including File Integrity Monitoring (FIM), IOC detection, brute-force attack detection, and network port scanning
Cybersecurity Python Tools

Projects included:

1. File Integrity Monitor
Detects file changes using SHA256 hashing like if u cahnged the spellings and created new file.

2. IOC Checker
Checks suspicious IP addresses.
Here I have created API keys by using AbuselPDB.

4. Brute Force Detector
Detects repeated failed login attempts.
Checks login attemps from user or IP addresses
If the same IP keeps entering the wrong password many times , it records it.

6. Port Scanner
Scans open ports on a target system.
Checks which ports on a computer or server are open,closed or filtered(block by firewall)
